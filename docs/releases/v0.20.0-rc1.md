# Release Notes - v0.20.0-rc1

### Sprint 19 — Editor Core
- Custom text rendering engine.
- Interactive minimap and document folding tree.
- Transactional history (Undo/Redo command logs).
- Hot workspace state recovery and diagnostics sidebar.

### Sprint 20 — Language Intelligence
- **20.1 & 20.2**: Language provider registry and hover tooltip resolution.

- **20.5**: Debounced completion panel overlay.
- **20.6**: Parameter hints and signature help popup overlay.
- **20.7**: Context quick fix lightbulb `💡` and code action menus.

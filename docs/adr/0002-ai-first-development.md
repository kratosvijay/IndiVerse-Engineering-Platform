# ADR 0002: AI-First Development Integration

- **Status**: Accepted
- **Date**: 2026-07-03
- **Author**: Platform Governance Board

## Context & Problem Statement
AI tools generate code rapidly, but without constraints, they introduce architectural violations and security risks.

## Decision Outcome
Define the **AI Constitution (AGENTS.md)** and provide metadata-enriched prompt assets that guide AI assistants during coding.

## Consequences
- **Positive**: Safer, high-fidelity AI-assisted generation.
- **Negative**: Prompts require ongoing maintenance.

# Prompt: Dart Unit Test Generator

- **Name**: Dart Unit Test Generator
- **Purpose**: Generates high-coverage unit tests for class logic.
- **Inputs**: Domain model or usecase code.
- **Outputs**: Test file verifying successes, validation errors, and boundary values.
- **Constraints**: Must use standard Dart/Flutter testing utilities and mock data.
- **Example**: Usecase `VerifyOtp`.
- **Expected Result**: `group()` test suite checking OTP validation boundaries.

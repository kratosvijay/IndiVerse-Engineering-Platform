# Prompt: Responsive Flutter Screen Generator

- **Name**: Responsive Flutter Screen Builder
- **Purpose**: Generates stateless Flutter screens bound to styling theme keys.
- **Inputs**: Screen layout specifications, controller state dependencies.
- **Outputs**: Flutter widget Dart file.
- **Constraints**: Zero state logic inside the widget. Adhere strictly to the theme layout variables.
- **Example**: `LoginScreen` with mobile/tablet layouts.
- **Expected Result**: Premium screen using Google Fonts, Outfit/Inter typography, and reactive binding layout.

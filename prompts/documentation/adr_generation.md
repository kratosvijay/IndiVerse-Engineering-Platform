# Prompt: ADR Proposal Generator

- **Name**: ADR Proposal Builder
- **Purpose**: Scaffold ADR documents from meeting notes or raw technical decisions.
- **Inputs**: Selected decision drivers, options evaluated, and final outcome.
- **Outputs**: Markdown record conforming to `templates/ADR.md`.
- **Constraints**: Must link downstream consequences cleanly.
- **Example**: Deciding on GetX over Bloc.
- **Expected Result**: Completed ADR document versioned for review.

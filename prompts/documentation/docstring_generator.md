# Prompt: Clean Code Docstring Generator

- **Name**: Clean Code Docstring Builder
- **Purpose**: Document classes, variables, and API responses systematically.
- **Inputs**: Raw code block.
- **Outputs**: Enriched code containing clean dartdocs or JSDocs comments.
- **Constraints**: Adhere strictly to language coding guidelines.
- **Example**: Untyped javascript API function.
- **Expected Result**: Documented endpoint showing inputs, outputs, and errors.
